ubuntu搭建mqtt开发play环境

Posted on 2019-12-30

我们用mosquitto作为broker，然后用paho mqtt c来作为客户端

部署mosquitto这个broker

  1. cd /home/ts/michaelma/2020/mqtt/
  2. git clone git@github.com:eclipse/mosquitto.git
  3. cd mosquitto
  4. mkdir build
  5. cd build
  6. cmake -DCMAKE_INSTALL_PREFIX=michaelma ..
  7. make
  8. make install
  9. cd michaelma/sbin
  10. export LD_LIBRARY_PATH=/home/ts/michaelma/2020/mqtt/mosquitto/build/michaelma/lib
  11. ./mosquitto -c /home/ts/michaelma/2020/mqtt/mosquitto/build/michaelma/etc/mosquitto/mosquitto.conf

开始play客户端

  1. cd /home/ts/michaelma/2020/mqtt/
  2. git clone git@github.com:eclipse/paho.mqtt.c.git
  3. cd paho.mqtt.c
  4. mkdir build
  5. cd build
  6. cmake -DCMAKE_INSTALL_PREFIX=michaelma -DPAHO_BUILD_SAMPLES=true -DPAHO_WITH_SSL=true ..
  7. make
  8. make install
  9. cd michaelma/bin
  10. export LD_LIBRARY_PATH=/home/ts/michaelma/2020/mqtt/paho.mqtt.c/build/michaelma/lib
  11. ./paho_c_sub -t hello -h localhost
  12. 再开一个终端，在里面执行./paho_c_pub -t hello -h localhost -m “wocao”
  13. paho_c_sub所在的终端会输出”wocao”

至此，play环境已经搭建好了

从上面的运行结果可以看出，只要满足mqtt协议，不同的broker和客户端都是能适配运行的。
这里使用的broker是mosquitto，其他的还有HiveMQ，不胜枚举
