gcc的参数

Posted on 2020-08-17

-Werrorshadow

当一个局部变量遮盖住了另一个局部变量，或者全局变量时，报错

-Waggregate-return

如果定义或调用了返回结构体或联合体的函数，编译器就发出警告

-fno-rtti

runtime type identification，这样编译器会为每个有虚函数的类添加一些信息以支持rtti特性

关闭对dynamic_cast和typeid的支持。如果你不需要这些功能，关闭它会节省一些空间。

-Werror=non-virtual-dtor

当一个类有多态性，而又没有虚析构函数时，发出错误

-Werror=return-type

函数走到了函数体最后却发现本应有返回值的地方没有返回值，报错

-Werror=format-security

编译器认为格式不安全，报错

-Werror=address

非法的比较，报错

-Werror=write-strings

将不可写的常量或指针赋值给可写的常量或指针，报错

# gcc
输出gcc/g++中的预定义宏
